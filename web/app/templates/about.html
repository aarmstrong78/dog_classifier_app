About the Dog Breed Classifier

This version of the web-app is hosted on Google App Engine ('GAE'), using the following platforms and tools

  • Flask for the main application and a prediction API (GAE 'endpoint')
  • gUnicorn serving up the two wsgi Flask apps
  • Firestore as the database engine
  • Google Cloud Storage to hold and serve the pictures
  • Docker to containerise the API with tensorflow, while the main app is on a standard GAE Python 3 runtime

The prediction model is Keras Xception, trained on the Stanford Dog Dataset supplemented with several hundred photos of Labradoodles.

An earlier version used the following platforms and tools

  • Flask for the main application and the prediction API
  • gUnicorn to serve up the two wsgi Flask apps
  • nginx web server to direct HTTP traffic
  • MySQL as the database engine
  • Docker to containerise each application for deployment

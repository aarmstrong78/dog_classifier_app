About the Dog Breed Classifier

This web-application uses the following platforms and tools

  • Flask for the main application and the prediction API
  • gUnicorn to serve up the two wsgi Flask apps
  • nginx web server to direct HTTP traffic
  • MySQL as the database engine
  • Docker to containerise each application for deployment
